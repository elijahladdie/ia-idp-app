Grant Types Reference

AUTHORIZATION_CODE

Standard OAuth 2.0 flow for web applications with server-side code.

Best for: Web applications, mobile apps

REFRESH_TOKEN

Allows applications to obtain new access tokens without user re-authentication.

Best for: Long-running applications

CLIENT_CREDENTIALS

Server-to-server authentication without user involvement.

Best for: API services, background jobs

Security Best Practices
